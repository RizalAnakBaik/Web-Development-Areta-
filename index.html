Sedang Latihan Music dan Video Player

Tag video HTML

Tag video digunakan untuk memasukan file vido ke dalam halaman web,
Tag ini memungkinkan pengguna untuk memutar file video langsung di halaman web tanpa harus membuka aplikasi pemutar video.

Tag Music HTML

Tag HTML audioo digunakan untuk memasukan file audi ke dalam halaman web,
Tag ini memungkinkan pengguna untuk memutar file musik langsung di halaman web tanpa harus membuka aplikasi pemutar video.
